Para rodar o pragrama completo é necessário compilar os arquivos .c na pasta lib. 

Exemplo:
´´´Windows
gcc -o main main.c lib/comprimento.c lib/converter_bits.c lib/converter_m_quadrado_cm_quadrado.c lib/converter_potencia.c lib/massa.c lib/temperatura.c lib/tempo.c lib/unidade_velocidade.c lib/unidadevolume.c

e depois executar o arquivo principal .\main.exe
